This project is an interactive dashboard developed to analyze teacher performance and operational metrics for MyNalanda.
It tracks key indicators such as attendance, late count, assignment corrections, and attrition risk. The dashboard provides visual insights to help monitor teacher performance and identify areas that need improvement. 
It enables better data-driven decision making for management.
